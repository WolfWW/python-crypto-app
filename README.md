## 依赖:
 - python3
 - pyDes,python-rsa,pyaes

第三方库安装：```pip -r requirements.txt```

## 使用方法（两种可选）：
 1. 代码下载后，双击运行crypto文件夹中的ginterface.py；
 2. 可使用pyinstaller将ginterface代码打包。打包方法可参考[我的打包记录](http://blog.csdn.net/elang6962/article/details/69259026)
